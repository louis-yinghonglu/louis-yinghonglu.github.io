Projects and Experiences

Bioinformatics @ UPenn

Did you know that learning about air quality can transform how students see the world? As the software engineering lead, I contribute to an edtech suite rooted in cutting-edge research. From firmware and app development to full-stack engineering, I bring this innovative project to life.

For this project, I utilized Python for firmware development, Kotlin for Android app creation, and Spring Boot with PostgreSQL for the backend system. I also leveraged AWS for cloud infrastructure, Docker for containerization, and Kubernetes for scalable deployment. Additionally, BLE communication and REST APIs ensured seamless interaction between devices and the cloud.

Augmented Reality @ Jorjin

Cutting-Edge Algorithms

Energy efficiency is crucial for delivering a great user experience in mobile augmented reality devices. I led R&D efforts utilizing a time-of-flight sensor to reduce energy consumption by over 50% compared to traditional methods.

Project and Team Management

At Jorjin, we emphasized transparent work distribution, efficient communication, and accountability. I championed the use of SOLID principles during code reviews to ensure clean, maintainable code. Our team also prioritized extensive documentation to streamline debugging and accelerate feature development.

Tech Stack

This project relied on a diverse set of technologies, including Python and C++ for developing core algorithms and computer vision features using OpenCV. On the mobile side, I worked with Java, Kotlin, and Android to create intuitive and efficient user interfaces. The project also involved advanced multithreading techniques to optimize performance and ensure smooth operation of real-time augmented reality functionalities.

Results

Over my 3-year tenure, our journey began with a brand-new hardware design and zero code. By the end, we achieved remarkable milestones, including winning the Best Choice Award and Golden Award at Computex. Our success led to investments by Foxconn and a collaboration with ASUS, establishing Jorjin as a leader in AR innovation.

Eye-Pressure and Retinography @ ITRI

We developed a novel at-home eye-pressure monitoring system to improve accessibility and convenience for patients. This system included an Android application designed for seamless user interaction and accurate data capture. Additionally, we streamlined the field screening process with a React web application, complemented by an Electron-based desktop solution for robust cross-platform functionality.

Tech Stack

The project utilized Android with Java for mobile development, React for building a responsive and user-friendly web application, and Electron for creating a desktop application that enhanced usability in clinical settings.

Results

The Industrial Technology Research Institute’s (ITRI) ICT Innovation for Self-Service Ophthalmoscope won the Merit Award in the “Innovative eHealth Solutions” category, showcasing the impact and innovation of this project.

Engineering Education Projects

Trebuchet Engineering Challenge: Physics in Action

In this hands-on project, we assembled a massive trebuchet and used it to play an oversized board game, teaching key principles of physics such as leverage, energy transfer, and projectile motion. This project allowed us to apply engineering concepts in an engaging and interactive way.

Line-Sensing Robot Car: Navigating Creativity

We designed a maze-solving robot car that follows a black line using an infrared sensor. This project emphasized teamwork, problem-solving, and creativity as we developed algorithms and collaborated to create a functional and responsive system.

The Golden Finger: Voice-Controlled Appliances

In this hackathon project, we created The Golden Finger, a system that lets you point to any home appliance and control it by shouting out voice commands. By combining computer vision and speech recognition, this project explored the power of user interfaces and accessibility.

Arduino Docs Translator: Bridging Language Gaps

To help non-English speaking developers, we built a Chinese translation website for the Arduino documentation, making technical knowledge more accessible and promoting learning across language barriers.

College Application Assistant: AI-Powered Guidance

We developed a website that uses GPT-3.5 to provide real-time suggestions and feedback for refining college application documents. This AI-powered assistant offers personalized guidance, helping students perfect their applications for a better chance of success.
